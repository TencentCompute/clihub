# Story 1.1: Configuration Schema and UI

**Epic**: Epic 1 - Custom CLI Arguments Configuration  
**Status**: Done 
**Created**: 2025-10-30  
**Assigned To**: Developer Agent  
**Story Points**: 3

---

## User Story

**作为** Codebuddy Terminal Editor 的用户  
**我希望** 通过 VS Code 设置配置 AI 工具的自定义命令行参数  
**以便** 无需修改扩展代码即可传递工具特定参数（如 `--yolo` 或 `--allow-all-tools`）

---

## Acceptance Criteria

1. ✅ **AC1**: 在 `package.json` 中添加配置模式，包含 `codebuddy.toolArguments` 设置
2. ✅ **AC2**: 设置 UI 允许为每个 AI 工具（codebuddy、gemini、claude、codex、copilot、cursor-agent）输入字符串
3. ✅ **AC3**: 配置接受有效的命令行参数字符串（例如 `--allow-all-tools --verbose`）
4. ✅ **AC4**: 空/未定义的配置默认为无自定义参数
5. ✅ **AC5**: 无效的参数格式显示清晰的验证消息（可选：在 Story 1.2 中实现运行时验证）
6. ✅ **AC6**: 现有工具配置不受影响

### Data Models

**ToolArgumentsConfig Interface** [Source: docs/architecture.md#Data Models]
```typescript
interface ToolArgumentsConfig {
  codebuddy?: string;
  gemini?: string;
  claude?: string;
  codex?: string;
  copilot?: string;
  'cursor-agent'?: string;
}
```

**Purpose**: 存储每个 AI 工具的自定义命令行参数  
**Integration**: 通过 VS Code 配置系统持久化，在终端创建时读取（Story 1.2）  
**Relationships**: 与 `AI_TOOLS` 数组的 `id` 字段映射

**Supported Tools** [Source: docs/architecture/tech-stack.md#External CLI Tools]:
- `codebuddy` - Codebuddy CLI（默认参数示例：`--dangerously-skip-permissions`）
- `gemini` - Gemini CLI（默认参数示例：`--yolo`）
- `claude` - Claude Code（默认参数示例：`--dangerously-skip-permissions`）
- `codex` - Codex（默认参数示例：`--full-auto`）
- `copilot` - GitHub Copilot CLI（默认参数示例：`--allow-all-tools`）
- `cursor-agent` - Cursor CLI（默认参数示例：`--force` 或 `-f`）

### Configuration Schema Design

### Coding Standards

**Configuration Schema Conventions** [Source: docs/architecture/coding-standards.md#VS Code Extension Patterns]:
- 使用 `markdownDescription` 支持富文本格式（换行、代码块）
- 使用 `order` 字段控制设置 UI 中的显示顺序
- 使用 `default` 字段提供默认值（避免 undefined）
- 使用 `description` 字段提供简短工具提示

**Naming Conventions** [Source: docs/architecture/coding-standards.md#Naming Conventions]:
- 配置键使用 `codebuddy.*` 命名空间
- camelCase 格式（如 `toolArguments`）
- 工具 ID 使用 kebab-case（如 `cursor-agent`）

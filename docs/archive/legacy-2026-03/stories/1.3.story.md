### Configuration Examples Format

**保守配置示例** (用于生产环境):
```json
{
  "codebuddy.toolArguments": {
    "codebuddy": "",
    "gemini": "",
    "claude": "",
    "codex": "",
    "copilot": "",
    "cursor-agent": ""
  }
}
```

**YOLO 模式配置示例** (用于个人开发):
```json
{
  "codebuddy.toolArguments": {
    "codebuddy": "--dangerously-skip-permissions",
    "gemini": "--yolo",
    "claude": "--dangerously-skip-permissions",
    "codex": "--full-auto",
    "copilot": "--allow-all-tools",
    "cursor-agent": "--force"
  }
}
```

**混合配置示例** (根据需要启用):
```json
{
  "codebuddy.toolArguments": {
    "codebuddy": "--dangerously-skip-permissions",
    "gemini": "--yolo",
    "claude": "",
    "codex": "",
    "copilot": "--verbose",
    "cursor-agent": "--force"
  }
}
```

### YOLO Mode Parameters Table

| AI 工具 | 参数 | 功能说明 | 风险等级 |
|---------|------|----------|----------|
| **Copilot** | `--allow-all-tools` | 允许访问所有工具和系统功能 | 🔴 高 |
| **Claude** | `--dangerously-skip-permissions` | 跳过权限确认，自动执行操作 | 🔴 高 |
| **Codebuddy** | `--dangerously-skip-permissions` | 跳过权限确认，自动执行操作 | 🔴 高 |
| **Codex** | `--full-auto` | 完全自动化模式，无需人工确认 | 🔴 高 |
| **Gemini** | `--yolo` | 自动确认所有操作 | 🔴 高 |
| **Cursor Agent** | `--force` 或 `-f` | 强制允许命令执行（除非明确拒绝） | 🔴 高 |

### Safety Warnings Content

**安全警告文本** (必须清晰显眼):
> ⚠️ **安全警告**: YOLO 模式参数会跳过安全检查或启用自动操作模式。这些参数允许 AI 工具在没有人工确认的情况下执行系统命令、修改文件、访问网络等操作。使用前请充分理解风险：
> - **高风险操作**: AI 可能执行危险命令（删除文件、修改配置、系统调用）
> - **数据安全**: 敏感数据可能在未经审查的情况下被访问或修改
> - **代码完整性**: 代码可能在未经审核的情况下被修改
> 
> **推荐使用场景**: 个人开发环境、实验项目、充分了解工具行为的情况下  
> **不推荐使用**: 生产环境、团队协作、处理敏感数据

### Troubleshooting Section

**常见问题和解决方案**:
1. **参数格式错误**: 确保使用正确的 JSON 格式，字符串需要双引号
2. **工具不识别参数**: 使用 `<tool-name> --help` 查看工具支持的参数
3. **参数未生效**: 关闭当前终端，重新打开以应用新配置
4. **多个参数**: 在单个字符串中用空格分隔，如 `"--flag1 --flag2 value"`
5. **特殊字符**: 包含空格或特殊字符的值需要引号，如 `"--arg 'value with spaces'"`
